🎯 Routing Logic

Entry: Supervisor Agent analyzes all queries

The supervisor uses LLM intelligence to decide which agent to call based on the query content. It can route to ordering, menu, grievance agents, or end the conversation.

📝 Example Query Paths
🍕 "I'd like to order a pizza"
ORDERING

Routed to Ordering Agent to handle the pizza order with size and toppings preferences.

🍽️ "What desserts do you have?"
MENU

Routed to Menu Agent to suggest dessert options and dish recommendations.

⚠️ "I received a cold meal"
GRIEVANCE

Routed to Grievance Agent to handle the complaint professionally and offer resolution.
